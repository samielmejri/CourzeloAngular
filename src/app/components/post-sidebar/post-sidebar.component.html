Creating a Post Instructions

  • Enter a title for your post in the Title field.
  • Provide a URL if applicable in the URL field.
  • Select the appropriate subreddit from the dropdown.
  • Compose your post content using the editor.
  • You can add an image or video using the file input.
  • Click the "Post" button to publish your post.
  • You can discard the post by clicking the "Discard" button.
